# **App Name**: Sweet Shop TDD

## Core Features:

- User Authentication: Register and log in users with token-based authentication.
- Sweet Listing: Display a list of all available sweets with their details, including price in Rupees. Include popular Indian sweets like Gulab Jamun (₹60), Rasgulla (₹50), and Jalebi (₹40).
- Sweet Search: Enable searching for sweets by name, category, or price range (in Rupees).
- Sweet Purchase: Allow users to purchase sweets, updating the quantity in stock. AI tool will update the purchase based on stock quantities.
- Sweet Restock: Enable admins to restock sweets, increasing the quantity in stock.
- Sweet Management: Admin users can add, update, and delete sweets.
- Role Management: Ensure users with the Admin role can only use the "delete" and "restock" endpoints.

## Style Guidelines:

- Primary color: Deep pink (#D43A7F) to reflect sweetness and appeal to user interest in dessert.
- Background color: Light pink (#F8E7F1) to provide a soft and inviting backdrop.
- Accent color: Light purple (#A44AFF) to highlight interactive elements.
- Headline font: 'Belleza', a sans-serif with personality, perfect for headlines. Body Font: 'Alegreya', a serif font suitable for body.
- Use relevant and visually appealing icons for sweet categories and actions.
- Responsive design for optimal viewing across devices.
- Subtle animations for adding items to the cart or updating inventory.
